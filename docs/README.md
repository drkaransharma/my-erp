# Documentation Hub

This folder contains all project documentation for the AI-augmented ERP platform that extends SAP and Oracle capabilities for business clients.

## Folder Structure

### 01-business-requirements/
Contains Business Requirements Documents (BRDs) that define the functional and business needs for each module of the platform. These documents serve as the foundation for development, ensuring every feature aligns with client expectations and enterprise workflows.

### 02-technical-docs/
Houses technical documentation covering system architecture, database schemas, API specifications, and integration connectors. This is the primary reference for developers and engineers building and maintaining the platform.

### 03-client-guides/
Provides end-user guides, getting-started tutorials, and standard operating procedures (SOPs) for clients and internal teams. These documents help users navigate the platform and onboard new clients efficiently.

### 04-progress-logs/
Tracks development progress through changelogs and dated log entries. This folder provides a historical record of what has been built, changed, or resolved over time.
