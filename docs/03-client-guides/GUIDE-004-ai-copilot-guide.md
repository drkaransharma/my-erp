# GUIDE-004: AI Copilot Guide

This guide explains how to use the platform's built-in AI Copilot, a conversational assistant that helps users query ERP data, generate reports, and perform actions using natural language. It covers how to ask questions about financial summaries, inventory levels, or customer records, how to refine AI-generated answers, and how the Copilot integrates with underlying SAP and Oracle data to deliver accurate, context-aware responses.
